# ☀️ Solê Recreação - Landing Page Profissional

### 📌 Descrição do Projeto
O projeto **Solê Recreação** é uma plataforma web desenvolvida para profissionalizar a presença digital de uma empresa de recreação infantil. O foco principal foi criar uma interface que equilibre o lúdico (essencial para o nicho) com a funcionalidade técnica, garantindo que os pais encontrem informações rápidas e um canal direto de orçamento.

A solução utiliza **Java com Spring Boot** para o backend, processando templates dinâmicos com **Thymeleaf**, e é distribuída via **Docker** para garantir consistência entre o desenvolvimento e a produção no **Render**.

### 🛠️ Tecnologias Utilizadas

* **Backend:** Java 17, Spring Boot 3.x
* **Frontend:** Thymeleaf, HTML5, CSS3 Moderno
* **DevOps/Deploy:** Docker, Render Cloud
* **Design:** Glassmorphism, CSS Animations, Mobile First
* **Ferramentas:** IntelliJ IDEA, Git, GitHub

---

### 📝 Padrão de Commits

O projeto segue o padrão **Conventional Commits**:
* `feat:` para novas funcionalidades (ex: adição do card de stats).
* `fix:` para correção de bugs (ex: ajuste de responsividade no menu).
* `style:` para alterações de design/CSS que não afetam a lógica.
* `chore:` para atualizações de build, dependências ou Dockerfile.
